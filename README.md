# VITScraped-npm-module
<center><h2>NPM module to fully scrape VTOP Student Login</h2></center>


Works for Winter as well as Fall Semester.

Enter vtop credentials, and get these things:

1. Marks
2. Assignment
3. Course page
4. Time Table
5. Course Details
6. Spotlight
7. Messages
8. CGPA
9. Attendance
10. Profile
11. Proctor
